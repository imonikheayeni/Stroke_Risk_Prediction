Stroke Prediction App

A working machine learning model to predict whether an individual may or may not be susceptible to a stroke. Machine learning is based on algorithms(calculations) that can learn from data without relying on 'rules-based' programming. The machine learns the calculations and trains on the large dataset to perform predictions. This final application result will only give an output of yes/no (binary: 1/0).

This application received 'real' private data from Kaggle. The data was cleaned, preprocessed, a model was then created, some machine learning algorithms were used to test the accurary of the model. Having obtained a good accuracy, the model was converted into a Pickle file and deployed with Flask (Flask is a Python web framework) and Github on Heroku Cloud platform.
