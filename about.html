Stroke Prediction App

A working machine learning model to test the prediction of whether an individual may or may not be susceptible to a stroke. Machine learning is based on algorithms(calculations) that can learn from data without relying on 'rules-based' programming. The machine learns the calculations and trains on the large dataset to perform the task. This final application result will only give an output of yes/no (binary: 1/0).

This application received 'real' private data from Kaggle. This information was put into a mongoDB Atlas for authors to review and create a machine learning model.
